# dbd-random-items
Примитивный рандомайзер для игры Dead By Daylight.

# Примечание: данная программа не является профессиональным ПО. Сделана буквально на коленях и имеет примитивный, банальный функционал.

Данная программа была разработана на досуге. Целью данной программы является добавление азарта во время онлайн-игры. 
Для работы программы её необходимо просто запустить, после этого она рандомно выдаст названия предметов, навыков и подношений.

P.S Была использована структура словаря, так как Dead By Daylight является онлайн-игрой, которая получает обновления и патчи. А значит что новые навыки добавляются со временем. Навыки используемые в программе, были актуальны на момент 07.03.2024. Так что, если хотите добавить или убрать, то дерзайте.
